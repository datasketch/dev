What are we doing

We are building a data science app store

We are building data visualization and analysis Shiny Apps that are (hopefully) very easy to use for non-tech users.

We embarked on a quest to make data science more accessible for non-coders. In 2019 2e launched a successful kickstarter campaign to democratize data visualization for people with non-programming experience. Nos after some months of development we are seeing live the first versions of our self-services data visualization and analysis tools.

Since then we have launched Datasketch Apps, an open source data visualization software as service, in this process we have been developing several R packages to help us build multiple data visualization apps at scale.
